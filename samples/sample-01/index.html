Customer Lookup MCP Server

Build an MCP server that lets Claude look up customer records by ID or name.

Starter branch — copy & use. This server already works with the included sample data. Run it, connect it to Claude Desktop, then swap in your own data. No API key needed — it uses your Claude.ai subscription.

Problem

Your team keeps customer data in a sheet or database and re-answers the same lookups all day. Build an MCP server that exposes a customer_lookup tool to Claude Desktop.

Tools it gives Claude

  • lookup_customer(id_or_name) — returns name, email, plan, status
  • list_customers() — returns a list of {id, name, plan}

Data source

Reads sample-data/customers.csv (8 example customers included). Replace the rows with your own to make it yours.

Steps

  1. Run pip install mcp
  2. (Optional) Run python test_tool.py — you should see "All tests passed"
  3. Edit claude_desktop_config.json so the path points to server.py on your computer
  4. Paste that config into Claude Desktop Settings > Developer, then quit and reopen Claude Desktop
  5. Ask Claude: "List all my customers"
Try this prompt in Claude Desktop: "List all my customers, then tell me which ones are not 'active' and what plan they're on."
