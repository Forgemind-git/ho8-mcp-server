Pricing Calculator MCP Server

Expose a calculate_quote tool that computes product quotes given a plan, seat count, and billing period — with annual discount logic built in.

Uses your Claude.ai subscription via Claude Desktop — no ANTHROPIC_API_KEY needed. Claude Desktop connects to this MCP server locally; your subscription provides the AI.

What this MCP server does

This server exposes a single tool called calculate_quote. It accepts three inputs — plan tier, number of seats, and billing period — and returns a detailed pricing breakdown including unit price, subtotal, discount, and annual savings.

The pricing table baked into server.py:

PlanMonthly / seatAnnual discountMax seats
Starter$12.000%5
Pro$29.0020% off50
Enterprise$79.0025% off10,000

No external API or database is needed — all logic runs locally.

Requirements

  • Python 3.10 or newer
  • Claude Desktop — free download at claude.ai/download (uses your Claude.ai subscription, no API key)
  • mcp Python package (installed below)

Build the server

Open a terminal and run:

# 1. Go into the sample directory
cd samples/sample-04

# 2. (Recommended) Create a virtual environment
python -m venv .venv
source .venv/bin/activate        # macOS / Linux
# .venv\Scripts\activate         # Windows PowerShell

# 3. Install dependencies
pip install -r requirements.txt

# 4. Verify the server starts
python server.py

You should see: Starting Pricing Calculator MCP server.... Press Ctrl+C to stop — Claude Desktop will launch it automatically when needed.

Connect to Claude Desktop

Open Claude Desktop → Settings → Developer → Edit Config and paste the following into claude_desktop_config.json. Replace the path with the absolute path to server.py on your machine.

{
  "mcpServers": {
    "pricing-calculator": {
      "command": "python",
      "args": ["/ABSOLUTE/PATH/TO/samples/sample-04/server.py"],
      "env": {}
    }
  }
}

Tips:

  • On macOS the path looks like /Users/yourname/ho8-mcp-server/samples/sample-04/server.py
  • On Windows use the full path with backslashes, e.g. C:\\Users\\yourname\\ho8-mcp-server\\samples\\sample-04\\server.py
  • If you used a virtual environment, replace python with the full path to the venv Python, e.g. /Users/yourname/ho8-mcp-server/samples/sample-04/.venv/bin/python
  • Save the file and restart Claude Desktop to load the new server.

Test it

After restarting Claude Desktop, type any of these prompts in a new conversation:

"What's the price for 10 Pro seats billed annually?"
"Give me a quote for 3 Starter seats, monthly billing"
"How much do I save going annual on Enterprise with 25 seats?"
"Compare monthly vs annual pricing for the Pro plan with 5 seats"

You should see Claude call the calculate_quote tool (visible as a tool-use block) and return a full pricing breakdown. If the tool icon appears in the chat toolbar, the server is connected correctly.
