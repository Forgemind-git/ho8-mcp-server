API Reference — Acme SaaS REST API v2

Authentication
--------------
All requests must include an Authorization header:
  Authorization: Bearer YOUR_API_KEY

API keys can be created at Settings > API Keys.

Base URL: https://api.acme.io/v2

Rate limits
-----------
Starter: 100 requests/minute
Pro: 1,000 requests/minute
Enterprise: 10,000 requests/minute

Exceeding the rate limit returns HTTP 429 Too Many Requests.
Retry after the number of seconds in the Retry-After header.

Endpoints
---------

GET /projects
  List all projects for the authenticated user.
  Returns: array of project objects.

POST /projects
  Create a new project.
  Body: { "name": "string", "description": "string" }
  Returns: project object.

GET /projects/:id
  Get a single project by ID.

DELETE /projects/:id
  Delete a project permanently. This action cannot be undone.

Webhooks
--------
Configure webhooks at Settings > Integrations > Webhooks.
Events: project.created, project.deleted, member.added, member.removed.
Payloads are signed with HMAC-SHA256 using your webhook secret.
