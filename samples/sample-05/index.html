Pricing Calculator MCP Server

Build an MCP server that lets Claude calculate quotes and commissions from your pricing table.

Week 4 M12 · MCP Solo Calculator
Starter branch — copy & use

This server already works as-is. No API key needed — it runs on your Claude.ai subscription through Claude Desktop. Just connect it and ask for a quote.

Problem

Sales reps keep pinging you for the same pricing numbers. Build an MCP server exposing your pricing and commission calculator.

Tools it gives Claude
  • get_quote(seats, plan, term_months) — returns {monthly, annual, discount}
  • get_commission(deal_value, tier) — returns {amount, pct}
Data source

Hardcoded pricing table in Python — no external database needed.

Steps
  1. Run pip install mcp
  2. (Optional) Run python test_tool.py to confirm the tools work — you should see "All tests passed."
  3. Edit the path in claude_desktop_config.json so it points to your copy of server.py
  4. Paste the pricing-calculator block into Claude Desktop → Settings → Developer → Edit Config, then save
  5. Quit Claude Desktop and reopen it
  6. Ask Claude for a quote (see the example prompt below)
Try this prompt: "Using my pricing tools, quote 10 seats on the pro plan billed annually (12 months), and tell me the senior-tier commission on that annual deal value."
