Doc Search MCP Server

Build an MCP server that searches a local folder of documents for Claude.

Starter branch. The files below are skeletons — open server.py and fill in the TODOs using Claude.ai.

Problem

Your important documents are scattered in a folder and finding the right one wastes time. Build an MCP server that searches them for Claude.

Tools to implement

  • search_docs(query) — returns [{filename, snippet, path}]
  • read_doc(filename) — returns {content}

Data source

Local folder of .txt / .md files. Add your documents to the sample-data/ folder.

Steps

  1. Add your .txt or .md files to sample-data/
  2. Open server.py and fill in each TODO
  3. Run pip install mcp then python server.py to test locally
  4. Paste the config from claude_desktop_config.json into Claude Desktop Settings > Developer
  5. Restart Claude Desktop and ask: "Search my docs for X"
